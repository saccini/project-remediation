# project-remediation

Hello and welcome to this project! The purpose of this repository is to host and document the digital remediation of the "Stöld" (Thievery) chapter from the historical Swedish legal code, *Sweriges rikes lag 1734*.

To fully understand this project, we highly suggest visiting our [GitHub Pages Website](https://saccini.github.io/project-remediation/), where you will be able to read about our archival methodology, the camera setup process, and the specific considerations and limitations we encountered while digitising a fragile physical artefact.

>[!NOTE]
> The source material used in this project is an 18th-century physical book. Please be aware that the original text features historical Swedish spelling, grammar, and typography. While our TEI-XML transcription strives for high fidelity to the original text, translating a physical, historical book into a digital, machine-readable format inherently requires interpretative choices. You can read more about the physical state of the artefact and our specific encoding decisions on the project website.
 
To explore this project, you do not need to install any specific software. Instead, you have three ways to interact with our work:

**1. The Website **
For the best reading and viewing experience, simply click the blue **View Website** badge at the top of this page. This will take you to our interactive site where you can explore the artefact, the TEI-XML transcriptions, and the embedded Dublin Core metadata in a user-friendly format.

**2. The Zenodo Archive**
If you are an archivist or researcher looking to download the raw, uncompressed high-resolution image files (.TIFF) or the standalone XML metadata files, please click the green **View Zenodo Archive** badge at the top of the page. 

**3. Cloning the Codebase**
The examples below are done in VS Code, but feel free to use whatever IDE you are comfortable with.

If you want to view the raw HTML, CSS, and XML files that make up this repository, you can clone it to your local machine. Open a new folder in VS Code, press “Clone Git Repository”, and then simply paste the URL from this GitHub repository into the search bar at the top.
