metadata

A comprehensive guide to the embedded parameters and external archival standards applied to the digital images in this repository.

Metadata is fundamentally "data about data." In the context of digital remediation, it is what transforms a simple photograph into a verifiable, archival asset. As Björk (2015) highlights, a digital reproduction is never a truly neutral copy of a physical artefact. It is heavily influenced by the technical and methodological choices of its creators.

To account for this, this project implements both traditional metadata and paradata, which is data that documents the actual process, environment, and tools of creation (Huvila et al., 2022). This page outlines our two distinct approaches to capturing this data:

First, we use Embedded Metadata (XMP/IPTC/EXIF), which is baked directly into every image file. By embedding our technical camera parameters, capture dates, and physical setup descriptions, we are actively preserving the paradata of our digitisation process. By embedding the data we ensure that it is never seperated from the visual file and thus preventing the creation of "orphan works"(Martinez & Terras, 2020).

Second, we use External Metadata based on the Dublin Core standard (DCMI, 2012). This acts as a project-wide bibliographic record, allowing the entire collection to be easily indexed, shared, and ingested by modern digital library systems.

Embedded metadata (XMP / IPTC / EXIF)

Added manually via GIMP → Image → Metadata → Edit Metadata to all structural images.

Xmp.dc.titleThe specific filename or identifier of the image (e.g., P1stold).
Xmp.dc.creatorErik Lundahl
Xmp.dc.descriptionA detailed, contextual description of the camera setup and the specific contents shown on the page or in the workspace.
Xmp.iptcExt.PersonInImageIdentifies any individuals visible in the frame during the process documentation.
Xmp.iptcExt.City
Xmp.iptcExt.CountryName
Xmp.iptcExt.CountryCode
Xmp.iptcExt.WorldRegion 		Structured geographic hierarchy marking the location of the digitisation (Stockholm, Sweden, SE, Europe).
Xmp.iptcExt.AddlModelInfoTechnical camera parameters (Aperture, ISO, Exposure, Megapixels) alongside the exact device age at the time of capture.
Exif.GPSInfo.GPSLongitude
Exif.GPSInfo.GPSLatitude
 Exact coordinates (Longitude & Latitude) of the workspace.
Xmp.dc.rightsThe copyright notice (e.g., (c) 2026 Sachini Nissanka and Erik Lundahl, licensed under CC-BY 4.0).
Xmp.xmpRights.WebStatementA direct link to the applied license (e.g., https://creativecommons.org/licenses/by/4.0/).
Xmp.xmpRights.MarkedBoolean flag indicating the image is copyrighted (True).
